## Overview

This project analyzes **regional sales data** to identify trends, patterns, and performance across different regions and product categories.

Using **Python for data analysis and Power BI for visualization**, the project aims to transform raw sales data into actionable business insights.

---

## Business Problem

Understanding sales performance is essential for businesses to improve strategy and revenue.

This project focuses on:

* Identifying **top-performing regions**
* Understanding **product category sales trends**
* Analyzing **revenue and profit performance**
* Discovering patterns that influence **business growth**

---

## Dataset
 The dataset contains sales information including:

Order Date
Region
Product Category
Sales Amount
Profit

The data was processed and analyzed using Python in Jupyter Notebook.
